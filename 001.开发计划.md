[快速开始]
1. 配置文件已创建（.env）：
   - 语言：中文（zh）
   - 端口：7861（避免冲突）
   - LM 后端：PyTorch（避免 Windows Triton 问题）
   - 模型：Turbo + 0.6B LM（适合 8GB 显存）

2. 启动服务：
   ```bash
   # 方式 1：使用启动脚本（推荐）
   start_gradio_ui.bat
   
   # 方式 2：使用命令行
   uv run acestep --language zh --port 7861
   ```

3. 打开浏览器访问：http://localhost:7861
   （注意：端口是 7861，不是 7860）
