## 🎵 核心功能使用

### 1. 生成中文歌曲

**简单模式**：
1. 选择"生成模式" → 简单
2. 输入描述：`一首温柔的中文流行情歌，钢琴伴奏`
3. 选择人声语言：`zh (Chinese)`
4. 点击"创建样本" → "生成音乐"

**自定义模式**：
1. 选择"生成模式" → 自定义
2. 填写详细描述和歌词
3. 设置 BPM、调性等参数
4. 点击"生成音乐"

### 2. Cover/Remix 翻唱

**跨语言翻唱示例**（英文 → 中文）：
1. 选择"生成模式" → Remix
2. 上传英文原曲
3. 描述：`Chinese Mandarin pop cover with emotional vocals`
4. 填入中文歌词
5. 人声语言：`zh`
6. Remix 强度：`0.7`
7. 翻唱强度：`0.2`
8. 点击"生成音乐"

### 3. 多语言控制

**中文普通话**：
- 人声语言：`zh` 或 `Chinese`
- 描述触发词：`Chinese Mandarin pop`, `国语流行`

**粤语**：
- 人声语言：`yue` 或 `Cantonese`
- 描述触发词：`Hong Kong Cantopop`, `粤语流行曲`

**英语**：
- 人声语言：`en` 或 `English`
- 描述触发词：`English pop`, `American rock`

## ⚙️ 技术细节

### Cover 功能实现

**核心参数**：
- `task_type`: `"cover"` 或 `"remix"`
- `audio_cover_strength`: 0.0-1.0（控制去噪步骤）
- `cover_noise_strength`: 0.0-1.0（旋律保留程度）
- `src_audio`: 源音频路径
- `caption`: 新风格描述
- `lyrics`: 新歌词（可选）

**推荐设置**：
- Remix 强度：0.6-0.7
- 翻唱强度：0.15-0.25（SFT 模型）
- 推理步数：50（SFT）或 8（Turbo）

### 语言控制实现

**方法 1：vocal_language 参数**
```python
vocal_language="zh"  # 中文
vocal_language="yue"  # 粤语
vocal_language="en"  # 英语
```

**方法 2：Caption 触发词**
```python
caption="Chinese Mandarin pop with emotional vocals"
caption="Hong Kong Cantopop ballad with Cantonese vocals"
caption="English rock with powerful vocals"
```

### i18n 框架

- 位置：`acestep/ui/gradio/i18n/`
- 语言文件：`zh.json`, `en.json`, `ja.json`, `he.json`
- 使用方法：`t("generation.tab_title")` → "🎵 生成"

## 🔧 配置建议

### GPU 配置

| GPU VRAM | 推荐模型 | LM 模型 | 其他设置 |
|----------|---------|---------|---------|
| ≤6GB | Turbo | 不启用 | INT8 量化 + CPU 卸载 |
| 6-8GB | Turbo | 0.6B (pt) | CPU 卸载 |
| 8-16GB | SFT | 1.7B (vllm) | 默认设置 |
| 16-24GB | SFT | 4B (vllm) | 默认设置 |
| ≥24GB | Base | 4B (vllm) | 全功能 |

### Cover 质量优化

**高质量设置**：
- 模型：SFT 或 Base
- 推理步数：50-100
- Remix 强度：0.6-0.7
- 翻唱强度：0.2-0.25
- 启用归一化

**快速预览**：
- 模型：Turbo
- 推理步数：4-8
- Remix 强度：0.5
- 翻唱强度：0.2

## 🐛 已知问题和限制

### 当前限制

1. **LoRA 与量化不兼容**：使用 LoRA 时需要禁用 INT8 量化
2. **Turbo 模型限制**：不支持引导比例（CFG）调整
3. **显存要求**：高质量生成建议 8GB+ VRAM

### 优化建议

1. **显存不足**：启用 CPU 卸载和 INT8 量化
2. **生成速度慢**：使用 Turbo 模型，减少批量大小
3. **发音不准**：使用 SFT 模型，明确设置人声语言

## 🌟 最佳实践

### 生成高质量音乐

1. **描述要详细**：包含流派、乐器、情绪、人声风格
2. **使用 SFT 模型**：质量明显优于 Turbo
3. **批量生成**：生成 2-4 个版本，选择最好的
4. **迭代优化**：根据结果调整参数，多次尝试

### Cover 翻唱技巧

1. **歌词匹配**：新歌词节奏与原曲相似
2. **参数起点**：Remix 0.6, Cover 0.2
3. **模型选择**：SFT 模型效果最好
4. **描述明确**：清楚说明目标风格和语言

### 多语言控制

1. **双重保险**：同时设置人声语言和描述触发词
2. **风格一致**：描述中的语言风格要与歌词匹配
3. **测试调整**：不同语言可能需要不同的参数
