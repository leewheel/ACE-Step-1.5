## ✨ 主要功能

### 🎵 音乐生成
- **简单模式**：用自然语言描述，AI 自动生成
- **自定义模式**：完全控制描述、歌词、BPM、调性等
- **批量生成**：一次生成 2-4 个版本，选择最好的

### 🎤 Cover/Remix 翻唱
- **风格转换**：将流行改成爵士、摇滚改成民谣
- **跨语言翻唱**：英文歌改成中文版，中文歌改成英文版
- **乐器重编**：钢琴伴奏改成吉他，原声改成电子
- **情绪调整**：欢快改成忧郁，慢歌改成快歌

### 🌍 多语言支持
- **中文普通话**：完美的国语发音
- **粤语**：地道的港式粤语
- **英语**：自然的英文演唱
- **50+ 语言**：支持日语、韩语、法语等

### 🎛️ 高级功能
- **LoRA 训练**：创建自己的音乐风格
- **音轨分离**：提取人声、鼓、贝斯等
- **音轨叠加**：为现有音乐添加新乐器
- **自动评分**：AI 评估生成质量
- **歌词时间戳**：自动生成 LRC 字幕

## 📖 使用教程

### 生成第一首中文歌

1. 选择"生成模式" → **简单**
2. 输入描述：
   ```
   一首温柔的中文流行情歌，钢琴伴奏，抒情动人
   ```
3. 选择"人声语言" → **zh (Chinese)**
4. 点击"创建样本"
5. 检查生成的描述和歌词
6. 点击"生成音乐"
7. 等待 10-15 秒，完成！

### 翻唱一首英文歌（改成中文版）

1. 选择"生成模式" → **Remix**
2. 上传英文原曲
3. 音乐描述：
   ```
   中文流行翻唱版本，保留原曲旋律，情感丰富的人声
   ```
4. 填入中文歌词（节奏与原曲相似）
5. 人声语言：**zh (Chinese)**
6. Remix 强度：**0.7**
7. 翻唱强度：**0.2**
8. 点击"生成音乐"

## 🎯 核心参数说明

### Cover/Remix 参数

**Remix 强度**（0.0-1.0）：
- **0.8-1.0**：非常接近原曲
- **0.6-0.7**：保留结构，明显风格变化（推荐）
- **0.4-0.5**：较大创意自由
- **0.1-0.3**：完全重新诠释

**翻唱强度**（0.0-1.0）：
- **0.0**：不保留旋律
- **0.15-0.25**：推荐范围（SFT 模型）
- **0.3-0.5**：强烈保留旋律
- **0.5-1.0**：最大保留

### 语言控制

**方法 1：人声语言下拉菜单**
- 中文：`zh` 或 `Chinese`
- 粤语：`yue` 或 `Cantonese`
- 英语：`en` 或 `English`

**方法 2：描述触发词**
- 中文：`Chinese Mandarin pop`, `国语流行`
- 粤语：`Hong Kong Cantopop`, `粤语流行曲`
- 英语：`English pop`, `American rock`

## 🔧 配置文件说明

配置文件位置：`.env`

**关键配置**：
```env
# 界面语言
LANGUAGE=zh

# 服务器端口
PORT=7861

# DiT 模型（turbo = 快速，sft = 高质量）
ACESTEP_CONFIG_PATH=acestep-v15-turbo

# LM 模型（0.6B = 8GB 显存，1.7B = 16GB 显存）
ACESTEP_LM_MODEL_PATH=acestep-5Hz-lm-0.6B

# LM 后端（pt = PyTorch，vllm = 更快但需要 Triton）
ACESTEP_LM_BACKEND=pt

# 是否启用 LM（Think 模式）
ACESTEP_INIT_LLM=true
```

## 💡 使用技巧

### 生成高质量音乐

1. **描述要详细**：
   ```
   ❌ 不好：流行歌曲
   ✅ 好：温柔的中文流行抒情歌，钢琴和弦乐伴奏，情感丰富的人声
   ```

2. **使用 SFT 模型**：
   - 在服务配置中选择 `acestep-v15-sft`
   - 推理步数设为 50

3. **批量生成**：
   - 批量大小设为 2-4
   - 生成多个版本，选择最好的

4. **迭代优化**：
   - 根据结果调整参数
   - 使用"将这些设置应用到 UI"功能

### Cover 翻唱技巧

1. **歌词匹配**：
   - 新歌词音节数与原曲相近
   - 重音位置与旋律匹配

2. **参数起点**：
   - Remix 强度：0.6
   - 翻唱强度：0.2
   - 根据结果调整

3. **模型选择**：
   - 快速预览：Turbo 模型，8 步
   - 高质量：SFT 模型，50 步

## 🐛 常见问题

### Q：界面显示英文而不是中文？
**A**：检查 `.env` 文件中 `LANGUAGE=zh` 是否正确设置

### Q：端口被占用？
**A**：在 `.env` 中修改 `PORT=8080`（或其他端口）

### Q：Triton 警告？
**A**：已自动使用 PyTorch 后端，不影响功能

### Q：显存不足？
**A**：在 `.env` 中设置 `ACESTEP_INIT_LLM=false` 禁用 LM

### Q：生成速度慢？
**A**：
- 使用 Turbo 模型
- 减少批量大小到 1
- 确认使用 GPU 而非 CPU

更多问题请查看：[故障排除指南.md](故障排除指南.md)
