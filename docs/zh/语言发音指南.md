# ACE-Step 1.5 多语言发音指南

## 概述

ACE-Step 1.5 支持 50+ 种语言的音乐生成，包括完美的中文普通话、粤语和英语发音。本指南将帮助你通过风格描述（Caption）和人声语言设置来精确控制发音效果。

## 🎤 语言控制方法

### 方法 1：人声语言设置（推荐）

在生成界面的"人声语言"下拉菜单中直接选择：

- **中文普通话**：选择 `zh` 或 `Chinese`
- **粤语**：选择 `yue` 或 `Cantonese`  
- **英语**：选择 `en` 或 `English`

### 方法 2：在描述（Caption）中使用风格触发词

在音乐描述中加入语言风格关键词，AI 会自动识别并应用相应的发音特征：

#### 中文普通话触发词
```
Chinese Mandarin pop
Mandarin ballad
Chinese pop with clear Mandarin pronunciation
普通话流行音乐
国语抒情歌曲
```

#### 粤语触发词
```
Hong Kong Cantopop
Cantonese pop ballad
Hong Kong style pop with Cantonese vocals
粤语流行曲
港式粤语情歌
```

#### 英语触发词
```
English pop
American rock with English vocals
British indie with clear English pronunciation
英式摇滚（英语演唱）
美式流行（英语人声）
```

## 🎵 Cover/Remix 功能的语言控制

使用 Cover 功能翻唱歌曲时，可以通过修改描述和人声语言来实现跨语言翻唱：

### 示例：将英文歌翻唱成中文版

1. **上传源音频**：原始英文歌曲
2. **生成模式**：选择 `Remix`
3. **音乐描述**：
   ```
   Chinese Mandarin pop cover with emotional vocals,
   keeping the original melody and structure
   ```
4. **歌词**：填入中文歌词（保持与原曲相似的节奏和结构）
5. **人声语言**：选择 `zh`
6. **Remix 强度**：0.6-0.8（保留原曲结构）
7. **翻唱强度（Cover Strength）**：0.15-0.25（保留旋律）

## 💡 高级技巧

### 1. 混合语言歌曲

在描述中同时提及多种语言：
```
Bilingual pop song mixing English and Chinese Mandarin,
switching between languages in verses and chorus
```

歌词示例：
```
[Verse 1 - English]
Walking down the street alone
Thinking about you all night long

[Verse 2 - 中文]
在这个城市的夜晚
我想念着你的笑脸

[Chorus - Mixed]
I can't let you go (不能让你走)
You're always in my heart (永远在我心中)
```

### 2. 方言和地域风格

除了标准普通话和粤语，还可以尝试其他中文方言风格：

- **台湾国语**：`Taiwanese Mandarin pop`
- **闽南语**：`Hokkien ballad`, `Taiwanese Hokkien song`
- **上海话**：`Shanghainese jazz`

### 3. 发音清晰度控制

在描述中强调发音清晰度：
```
Clear and articulate Mandarin pronunciation
Crystal clear Cantonese vocals
Precise English diction
```

## 🔧 故障排除

### 问题 1：生成的语言不对

**解决方案**：
1. 确保"人声语言"设置正确
2. 在描述中明确添加语言关键词
3. 检查歌词是否使用了正确的语言文字

### 问题 2：发音不够自然

**解决方案**：
1. 在描述中添加更多风格细节（如"自然的"、"流畅的"）
2. 调整 LM Temperature（0.8-1.2 之间）
3. 尝试生成多个版本（批量大小 2-4），选择最好的

### 问题 3：Cover 功能语言转换效果不佳

**解决方案**：
1. 使用 SFT 模型而非 Turbo 模型
2. 调整翻唱强度（Cover Strength）到 0.15-0.25
3. 确保新歌词的节奏与原曲匹配
4. 适当调整 Remix 强度（0.5-0.7）
