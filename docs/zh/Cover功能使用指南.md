# ACE-Step 1.5 Cover/Remix 功能完全指南

## 🎤 什么是 Cover 功能？

Cover 功能（也称为 Remix）允许你将现有歌曲转换成不同的风格，同时保留原曲的旋律和结构。这类似于 Suno 的 Cover 功能，可以实现：

- 🎵 **风格转换**：将流行歌曲改编成爵士、摇滚、电子等任何风格
- 🌍 **跨语言翻唱**：将英文歌改编成中文版，或将中文歌改编成英文版
- 🎸 **乐器重编**：改变伴奏乐器，如将钢琴伴奏改成吉他
- 🎭 **情绪调整**：将欢快的歌曲改编成忧郁版本，或反之

## 🚀 快速开始

### 基础 Cover 流程

1. **选择生成模式**：在界面上选择 `Remix` 模式
2. **上传源音频**：点击"源音频"区域上传你要翻唱的歌曲
3. **编写新描述**：在"音乐描述"中写下你想要的新风格
4. **修改歌词**（可选）：如果要跨语言翻唱，填入新语言的歌词
5. **调整参数**：
   - **Remix 强度**：0.5-0.8（控制与原曲的相似度）
   - **翻唱强度**：0.15-0.25（控制旋律保留程度）
6. **点击生成**：等待 AI 生成你的 Cover 版本

## 📖 详细教程

### 场景 1：风格转换（保持语言不变）

**目标**：将一首流行歌曲改编成爵士风格

#### 步骤：

1. **上传源音频**：原始流行歌曲（MP3/WAV/FLAC）

2. **音乐描述**：
   ```
   Smooth jazz cover with sultry vocals, 
   featuring piano trio, walking bass, and brush drums,
   intimate and sophisticated atmosphere
   ```

3. **歌词**：保持原歌词不变（或留空让 AI 自动识别）

4. **人声语言**：与原曲相同（如原曲是中文，选择 `zh`）

5. **参数设置**：
   - Remix 强度：`0.6`（保留较多原曲结构）
   - 翻唱强度：`0.2`（保留旋律）
   - 推理步数：`8`（Turbo 模型）
   - BPM：留空或设置为 `Auto`

6. **点击"生成音乐"**

### 场景 2：跨语言翻唱

**目标**：将英文歌曲翻唱成中文版

#### 步骤：

1. **上传源音频**：英文原曲

2. **音乐描述**：
   ```
   Chinese Mandarin pop cover with emotional vocals,
   keeping the original melody and chord progression,
   modern production with piano and strings
   ```

3. **歌词**：填入中文歌词
   ```
   [主歌]
   在这个寂静的夜晚
   我想起了你的笑脸
   那些美好的时光
   如今只剩下回忆
   
   [副歌]
   如果能够重来一次
   我会紧紧抓住你的手
   不让你离开我的世界
   永远陪在你身边
   ```

4. **人声语言**：选择 `zh`（中文）

5. **参数设置**：
   - Remix 强度：`0.7`（较高，保持原曲结构）
   - 翻唱强度：`0.2`（保留旋律）
   - 使用 SFT 模型（推荐，效果更好）
   - 推理步数：`50`（SFT 模型）

6. **点击"生成音乐"**

**提示**：
- 确保中文歌词的音节数量与原曲相近
- 可以先用 AI 工具将英文歌词翻译成中文，再调整节奏
- 多生成几个版本（批量大小 2-4），选择最好的

## ⚙️ 参数详解

### Remix 强度（Remix Strength）

控制输出与原曲结构的相似度：

- **0.8-1.0**：非常接近原曲，只有细微的风格变化
- **0.6-0.7**：保留原曲结构，但有明显的风格转换（推荐起点）
- **0.4-0.5**：较大的创意自由，可能有显著变化
- **0.1-0.3**：非常自由的重新诠释，可能与原曲差异很大

### 翻唱强度（Cover Strength / Melody Retention）

控制旋律保留程度：

- **0.0**：完全不保留旋律，纯风格迁移
- **0.1-0.25**：推荐范围（使用 SFT 模型时），保留旋律但允许风格变化
- **0.3-0.5**：强烈保留旋律，风格变化受限
- **0.5-1.0**：最大程度保留旋律，几乎不变

**重要提示**：
- 使用 **SFT 模型**时，翻唱强度 0.1-0.25 效果最好
- 使用 **Turbo 模型**时，可能需要调整到 0.3-0.5
- 如果旋律保留不够，提高翻唱强度
- 如果风格转换不够明显，降低翻唱强度

### 推理步数（Inference Steps）

- **Turbo 模型**：4-8 步（推荐 8）
- **SFT/Base 模型**：50-100 步（推荐 50）

更多步数 = 更高质量，但生成时间更长

## 🎯 最佳实践

### 1. 选择合适的模型

- **快速预览**：使用 Turbo 模型，8 步
- **高质量 Cover**：使用 SFT 模型，50 步
- **最大控制**：使用 Base 模型，100 步 + 引导比例

### 2. 描述要具体

❌ 不好的描述：
```
Jazz version
```

✅ 好的描述：
```
Smooth jazz cover with sultry female vocals,
featuring piano trio, walking bass, and brush drums,
intimate late-night atmosphere, sophisticated arrangement
```

### 3. 歌词匹配

跨语言翻唱时，确保：
- 新歌词的音节数量与原曲相近
- 重音位置与原曲旋律匹配
- 保持相似的情感表达

### 4. 迭代优化

1. 先用默认参数生成一个版本
2. 根据结果调整：
   - 旋律不够准确 → 提高翻唱强度
   - 风格不够明显 → 降低 Remix 强度，优化描述
   - 结构不对 → 提高 Remix 强度
3. 生成多个版本（批量大小 2-4）对比选择

### 5. 使用参考音频

如果你有理想的风格参考，可以同时上传：
- **源音频**：要翻唱的歌曲
- **参考音频**：风格参考（可选）

## 🔧 常见问题

### Q1：生成的 Cover 旋律不准确

**解决方案**：
1. 提高翻唱强度到 0.25-0.35
2. 使用 SFT 模型而非 Turbo
3. 提高 Remix 强度到 0.7-0.8
4. 确保歌词节奏与原曲匹配

### Q2：风格转换不够明显

**解决方案**：
1. 降低 Remix 强度到 0.4-0.5
2. 在描述中添加更多风格细节
3. 明确指定乐器和编曲风格
4. 尝试降低翻唱强度到 0.1-0.15

### Q3：跨语言翻唱效果不自然

**解决方案**：
1. 确保"人声语言"设置正确
2. 在描述中明确提及目标语言（如 "Chinese Mandarin"）
3. 调整歌词使其更符合目标语言的发音习惯
4. 使用 SFT 模型，推理步数 50+

### Q4：生成的音频有杂音或失真

**解决方案**：
1. 启用"归一化"功能
2. 调整 Latent Shift 到 -0.04
3. 调整 Latent Rescale 到 0.91
4. 确保源音频质量良好（推荐 WAV 或 FLAC）

### Q5：生成时间太长

**解决方案**：
1. 使用 Turbo 模型（8 步，约 10 秒）
2. 减少批量大小到 1-2
3. 启用 Flash Attention（如果可用）
4. 使用 GPU 而非 CPU
