# ACE-Step 1.5 快速开始指南（中文）

## 🚀 5 分钟快速部署

### 第一步：安装 uv 包管理器

**Windows 用户**：
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

**Linux/macOS 用户**：
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### 第二步：配置中文界面

1. 复制环境配置文件：
```bash
cp .env.example .env
```

2. 编辑 `.env` 文件，确保语言设置为中文：
```
LANGUAGE=zh
```

### 第三步：安装依赖

```bash
uv sync
```

这个命令会自动安装所有需要的依赖包，包括 PyTorch、Gradio 等。

### 第四步：启动服务

**方式 1：使用命令行**
```bash
uv run acestep
```

**方式 2：使用启动脚本**
- Windows：双击 `start_gradio_ui.bat`
- Linux/macOS：运行 `./start_gradio_ui.sh`

### 第五步：打开浏览器

启动成功后，打开浏览器访问：
```
http://localhost:7860
```

你将看到完整的中文界面！

## 🎵 快速生成第一首歌

### 方法 1：简单模式（推荐新手）

1. 在界面上选择"生成模式" → **简单**
2. 在"歌曲描述"中输入：
   ```
   一首温柔的中文流行情歌，钢琴伴奏，抒情动人
   ```
3. 选择"人声语言" → **zh (Chinese)**
4. 点击"创建样本"
5. 检查生成的描述和歌词
6. 点击"生成音乐"
7. 等待 10-30 秒，你的第一首 AI 音乐就生成了！

### 方法 2：自定义模式（完全控制）

1. 选择"生成模式" → **自定义**
2. 填写"音乐描述"：
   ```
   一首温柔的中文流行抒情歌曲，以钢琴和弦乐为主，
   人声情感丰富，温暖动人的氛围
   ```
3. 填写"歌词"：
   ```
   [主歌]
   在这个寂静的夜晚
   我想起了你的笑脸
   那些美好的时光
   如今只剩下回忆
   
   [副歌]
   如果能够重来一次
   我会紧紧抓住你的手
   不让你离开我的世界
   永远陪在你身边
   ```
4. 设置参数：
   - 人声语言：**zh (Chinese)**
   - BPM：**80**
   - 调性：**C Major**
   - 时长：**30** 秒
5. 点击"生成音乐"

## 🎤 使用 Cover 功能翻唱歌曲

### 场景：将一首英文歌翻唱成中文版

1. 选择"生成模式" → **Remix**
2. 上传源音频（英文原曲）
3. 填写"音乐描述"：
   ```
   中文流行翻唱版本，保留原曲旋律和结构，
   情感丰富的人声，现代流行编曲
   ```
4. 填写中文歌词（节奏与原曲相似）
5. 选择"人声语言" → **zh (Chinese)**
6. 设置参数：
   - Remix 强度：**0.7**
   - 翻唱强度：**0.2**
   - 推理步数：**50**（使用 SFT 模型）
7. 点击"生成音乐"

**详细教程**：查看 `docs/zh/Cover功能使用指南.md`

## ⚙️ 服务配置建议

### 首次使用配置

1. 点击"服务配置"标签
2. 选择模型：
   - **快速预览**：`acestep-v15-turbo`
   - **高质量**：`acestep-v15-sft`
3. 选择 5Hz LM 模型（根据你的 GPU）：
   - **6-8GB VRAM**：`acestep-5Hz-lm-0.6B`
   - **8-16GB VRAM**：`acestep-5Hz-lm-1.7B`
   - **16GB+ VRAM**：`acestep-5Hz-lm-4B`
4. 勾选"初始化 5Hz LM"（启用 Think 模式）
5. 点击"初始化服务"
6. 等待状态显示"✅ 服务已初始化"

### GPU 显存不足？

如果你的 GPU 显存较小（<8GB），建议：
- 启用"闲置时转移到 CPU"
- 启用"INT8 量化"
- 使用 Turbo 模型
- 批量大小设为 1

## 🔧 常见问题

### Q1：启动时提示找不到模型

**解决方案**：首次启动时，模型会自动下载。请确保网络连接正常，等待下载完成。

### Q2：生成速度很慢

**解决方案**：
1. 使用 Turbo 模型（8 步，约 10 秒）
2. 启用 Flash Attention（如果可用）
3. 确保使用 GPU 而非 CPU
4. 减少批量大小

### Q3：生成的中文发音不准确

**解决方案**：
1. 确保"人声语言"设置为 `zh`
2. 在描述中添加 `Chinese Mandarin` 关键词
3. 使用 SFT 模型而非 Turbo
4. 调整 LM Temperature 到 0.8-1.2

### Q4：Cover 功能旋律不准确

**解决方案**：
1. 提高翻唱强度到 0.25-0.35
2. 使用 SFT 模型
3. 提高 Remix 强度到 0.7-0.8
4. 确保歌词节奏与原曲匹配

## 💡 推荐工作流程

### 新手推荐流程

1. **第一次尝试**：使用简单模式，让 AI 自动生成
2. **学习参数**：观察生成的描述、歌词、BPM 等
3. **切换自定义**：开始手动调整参数
4. **尝试 Cover**：上传喜欢的歌曲进行翻唱
5. **训练 LoRA**：创建自己的音乐风格

### 专业用户流程

1. **精确控制**：使用自定义模式，手动设置所有参数
2. **批量生成**：批量大小 2-4，生成多个版本对比
3. **使用 LoRA**：加载训练好的 LoRA 适配器
4. **API 调用**：使用 REST API 进行批量处理
5. **后期处理**：导出 FLAC 格式，使用 DAW 进一步编辑
